# Java Build System

## Clone the Repository
```bash
git clone https://github.com/X-RugvedCodes-X/Java-Program-Build-System.git .
```

## Run the Setup Script
```bash
./setup
```

## Build the Project (After Changes)
```bash
./build main # ./build <package-name>
```

## For Just Running the Project
```bash 
./run main # ./run <package-name>
```

## Configure Class Path in VS Code (Ignore if Already Done) 
```text
Ctrl + Shift + P -> Java: Configure ClassPath -> Set the 'src' and 'out' Directories Appropriately
```
